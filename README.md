# 🌊 TideULT

**The all-in-one media server that brings everything to shore.**

TideULT is a self-hosted media platform built for people who want a complete, unified experience without the hassle of configuring and maintaining five separate services. Stream your library, find new content, download it automatically, and organize it all from one clean interface. Just set it up once, and let the tide do the rest.

---

## Features

- **Media Server** — Stream your movies, TV shows, and music to any device on your network or remotely, with full transcoding support powered by FFmpeg
- **Built-in Indexer Management** — Add and manage torrent indexers directly inside the app, no Prowlarr setup required
- **Automated Media Tracking** — Set up shows and movies you want, and TideULT will find and grab them automatically as they become available
- **Integrated BitTorrent Client** — A fully built-in torrent client so you never need to install qBittorrent or Deluge separately
- **Smart File Organizer** — Automatically sorts, renames, and moves downloaded files into your library the right way, every time
- **Modular Setup Panel** — Every major feature is optional. Turn on only what you need — media server only, full arr stack, or everything at once
- **Unified Dashboard** — One interface to rule them all. No more juggling tabs between Jellyfin, Sonarr, Radarr, Prowlarr, and a torrent client
- **User Management** — Multi-user support with individual profiles, permissions, and watch history
- **Real-Time Download Progress** — Watch your downloads come in live right from the dashboard
- **Mobile Friendly** — Fully responsive UI that works great on phones and tablets

---

## Built With

- **Python** — Backend, automation, torrent engine, and indexer logic
- **FastAPI** — High-performance API layer
- **React + TypeScript** — Modern, responsive frontend
- **FFmpeg** — Transcoding and media processing

---

## Philosophy

TideULT is built on one idea: **your media setup should just work.** The self-hosted community has incredible tools, but getting them all talking to each other is a weekend project in itself. TideULT brings it all under one roof so you can spend less time configuring and more time watching.

Every feature is optional. Every module is toggleable. You're always in control.
